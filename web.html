About the Project

Hand-drawn sketch recognition is a challenging task due to the abstract and minimalistic nature of sketches compared to natural images. This project aims to classify sketches into predefined object categories using various machine learning techniques. The significance of this research lies in its applications in human-computer interaction and content-based image retrieval. We explore different traditional feature extraction techniques and compare their effectiveness in conjunction with machine learning models.
